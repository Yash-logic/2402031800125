## 📌 Project Overview

FinWise is a comprehensive, all-in-one financial management solution designed to help users track expenses, set budgets, and gain deep analytical insights into their spending habits.

This project includes:

* 🎨 High-fidelity Figma Prototype
* 💻 Fully responsive Web Application

---

## 🚀 Features

### 1. 🎨 Prototype Design (Figma)

* 10 Essential Screens (Login → Dashboard → Analytics)
* Light & Dark Mode UI
* Security Screens (PIN & Fingerprint)
* 80+ UI Components for consistency

---

### 2. 💻 Web Development

* 📊 Dynamic Dashboard (Balance, Income, Expense)
* 📈 Interactive Charts (Chart.js)
* 🧾 Transaction Management (jQuery table)
* 📱 Responsive UI (Bootstrap 5)
* 🌙 Dark Mode Toggle
